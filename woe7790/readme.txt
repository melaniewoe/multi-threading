*******************************************************
*  Description of the program
*******************************************************

Multithreading_pthread is a program that can perform matrix multiplication using Pthreads. The program will have one thread per row of the product matrix. Each set of data processed is labeled as matrix A and matrix B. The result of matrix multiplication will be stored as matrix C. After the program completed the matrix multiplication, it will also tell the user of the execution time.

*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program.  The file reads the file “matrix.txt”. 
It computes the matrix multiplication based on number 
of threads. It prints out the result matrix C and total 
execution time.
